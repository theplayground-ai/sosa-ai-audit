Part 1: The Basics
Slash Commands You Need to Know
These are your daily drivers. Memorize these and you'll move 10x faster.
/init
Initialize a new project. Creates CLAUDE.md, sets up context structure, gets Claude oriented to your codebase.
$ /init
Use when: Starting any new SOSA project or tool
/compact
Compress conversation history to free up context window. Essential for long sessions. Claude keeps the important stuff, drops the noise.
$ /compact
Use when: Claude starts forgetting things or slowing down
/cost
Check how much this session has cost so far. Track your API usage and know when to compact.
$ /cost
Use when: Want to track spend on a project
/clear
Start fresh. Wipes conversation history but keeps your CLAUDE.md context. Good for switching tasks.
$ /clear
Use when: Switching from content work to operations
/help
See all available commands and how to use them. Your quick reference when you forget a command.
$ /help
Use when: Can't remember a command name
/model
Switch between Claude models. Opus for complex strategy, Sonnet for fast coding, Haiku for quick tasks.
$ /model sonnet
Use when: Simple tasks don't need the biggest model
/vim
Toggle vim mode for keyboard-driven editing. For power users who want to move faster in the terminal.
$ /vim
Use when: You're comfortable with terminal workflow
/permissions
Control what Claude can do automatically vs. what needs your approval. Set trust levels for different actions.
$ /permissions
Use when: Setting up a new workspace
/doctor
Diagnose issues with your Claude Code setup. Checks API keys, permissions, and configuration.
$ /doctor
Use when: Something isn't working right
Part 2: The Real Power
Custom Skills: Your AI Automations
Skills are reusable prompts that run entire workflows with one command. This is where Claude Code goes from assistant to operations team.
HOW SKILLS WORK
1
Create a .md File
Write instructions in a markdown file inside .claude/commands/
2
Define the Prompt
Tell Claude exactly what to do, what context to read, and what to output
3
Run with /command
Type /your-skill-name and Claude executes the entire workflow
4
Iterate + Improve
Refine the skill based on results. Skills get better the more you use them

Expedition Email Writer

One command generates a complete post-expedition email sequence customized per trip destination, with personalized impact metrics and next-trip teasers.

Skill File: .claude/commands/expedition-email.md
# Expedition Email Writer Read context/expeditions.md for trip details. Read context/partners.md for NGO impact data. Generate a 5-email post-expedition drip: 1. Welcome home (send immediately) 2. Photo gallery (day 3) 3. Impact report with real numbers (day 7) 4. Next trip early access (day 14) 5. Alliance membership invite (day 30) Use Blake's voice. Authentic, not salesy. Include specific impact metrics per partner.
ExpeditionsEmail

Content Repurposer

Takes one piece of long-form content (expedition vlog, podcast episode, blog post) and generates versions for every platform.

Skill File: .claude/commands/repurpose.md
# Content Repurposer Input: $ARGUMENTS (paste transcript or content) Generate for each platform: - Instagram caption (max 2200 chars, 5-10 hashtags) - TikTok script (60 sec hook + story) - Twitter thread (5-7 tweets) - LinkedIn post (professional angle) - Blog post (800-1200 words, SEO optimized) - YouTube description + timestamps - Newsletter snippet (200 words) Match SOSA's voice: urgent but warm, mission-driven, never preachy.
ContentAll Platforms

Blog Post Generator

Transforms field notes, expedition journals, or interview transcripts into SEO-optimized blog posts for the SOSA Adventures blog.

Skill File: .claude/commands/blog.md
# SOSA Blog Writer Input: $ARGUMENTS (field notes or transcript) Write a 1000-1500 word blog post for thesosa.com/blogs/sosa-adventures Include: - Compelling headline (under 70 chars) - Hook paragraph that pulls readers in - 3-5 subheadings for scannability - Specific details (species names, locations) - Call to action (expedition booking or merch) - SEO keywords naturally woven in Tone: first-person Blake voice. Raw and real, not polished corporate.

Video Show Notes Generator

Takes a video transcript and generates complete show notes, timestamps, key quotes, and social clips list for the video team.

Transcription with Whisper

Transcribe any video or audio file locally using OpenAI's Whisper model. Free, unlimited, runs on your machine. Always use "medium" model for accuracy.

whisper video.mp4 --model medium

Auto-Generate Timestamps

Feed a transcript to Claude and get clean timestamps for YouTube descriptions. Identifies topic changes, key moments, and natural break points.

/show-notes [paste transcript]

Clip Identification for Reels

Claude reads the transcript and flags the most shareable 30-60 second segments. Looks for emotional moments, surprising facts, and quotable lines.

Automated via show-notes skill

FFmpeg Audio Mixing

Add background music to videos with proper volume fading. Claude generates the exact FFmpeg command for your audio specs. No editing software needed.

ffmpeg -i video.mp4 -i music.mp3 ...

Batch Caption Generation

Generate Instagram captions, TikTok scripts, and Twitter threads from a single transcript. One input, seven platform outputs.

/repurpose [paste transcript]

Content Calendar Planning

AI plans your posting schedule based on expedition dates, awareness days, and optimal engagement times. Fills gaps automatically.

/content-calendar [month]

SEO Title + Description Writer

Claude generates YouTube titles optimized for search, plus full descriptions with keywords, links, and calls to action. A/B test options included.

/youtube-seo [topic]

Podcast Production

"Boots on the Ground" podcast pipeline: transcribe, generate show notes, create audiograms, write newsletter summaries, suggest guests.

/podcast-pipeline [episode file]

Thumbnail Text Generator

3 thumbnail text options per video, optimized for click-through rate. Short, punchy, curiosity-driven. Based on what performs in the conservation niche.

Included in show-notes output
